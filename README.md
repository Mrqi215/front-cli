# vue-front-cli

前端脚手架

[开发清单](doc/开发清单.md)

[项目文档](doc/项目文档.md)

## 🌟项目依赖初始化
```
yarn
```

### 🌟项目运行
- 第一个项目
```
npm run d firstProject
cnpm run d firstProject
yarn d firstProject
```

- 第二个项目
```
npm run d secondProject
cnpm run d secondProject
yarn d secondProject
```

### 🌟项目打包
```
yarn b 项目
```

### 提交规范
```
yarn cz
```
